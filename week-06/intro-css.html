Cascading Style Sheet (CSS)

CSS is used to style and HTML document. CSS define how HTML elements / document is displayed. Also, CSS can be used for styling the print version of the HTML document.

How to include CSS in HTML document

  1. External CSS - This is the proper way.
  2. Internal CSS - in the head part of the HTML document, avoid this practise
  3. Inline CSS - avoid this practise

CSS selectors

You have 5 options on how to use CSS selectors:

  • Use CSS element selector
  • Use CSS class selector
  • Use CSS ID selector
  • Use CSS group selector
  • Use CSS universal selector - mainly used for reset HTML document and debugging.

CSS element selector

Example: p { color: darkblue; }

Here the "p" CSS element selector will style ALL paragraphs in entire HTML document. All paragraphs will inherit darkblue colour.

CSS class selector

Example: .important { color: red; font-weight: bold; }

The CSS class starts with "." the full stop at the beginning of it's name in the CSS language. The name of the class is your choice, and it should be something meaningful! The class can be called on multiple HTML elements. Please notice that when you call the class on HTML opening element "." the full stop in NOT used.

CSS ID selector

Example: #sub-chapter-id { color: white; background-color: goldenrod; font-family: "Courier New", Courier, monospace; font-size: larger; }

The CSS ID starts with "#" the hashtag at the beginning of it's name in the CSS language. The name of the ID is your choice, and it should be something meaningful! You can use ID name ONLY ONCE in HTLM document as it is Unique Identifier.
